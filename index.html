Crystal Collector!

You will be given a random number at the start of the game

There are four crystals below. By clicking a crystal you will add a specific amount of points to your total score.

You win the game by matching your total score to the random number, you lose the game if yoru total score goes above the randome number.

The value of each crystal is hidden from you until you click on it.

Each time a new game starts, the game will change the value of each crystal
